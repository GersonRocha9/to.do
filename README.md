As funcionalidades do aplicativo são:

Contagem de tarefas;
Adicionar uma nova tarefa;
Remover uma tarefa;
Marcar e desmarcar uma tarefa como concluída.

```bash
git clone https://github.com/GersonRocha9/todos.git
```

```bash
yarn install
```
